# Pre-commit Checklist

提交或结束任务前，确认：

- [ ] 功能已完成或明确标记为 BLOCKED。
- [ ] 已运行测试 / lint / type-check。
- [ ] 已记录验证命令和结果。
- [ ] 已更新 `.harness/state/feature_list.json`。
- [ ] 已更新 `.harness/logs/progress-log.md`。
- [ ] 如有重要决策，已更新 `.harness/decisions/decision-log.md`。
- [ ] 已更新 `.harness/state/session-handoff.md`。
- [ ] 没有提交 `.env`、密钥、token、cookie、私钥。
