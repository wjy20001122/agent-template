# Progress Log

> 用途：让人和 AI Agent 都能快速知道项目进展、决策和阻塞点。

## 记录格式

### {{YYYY-MM-DD}} - {{TASK_TITLE}}

#### Done
- 完成了什么。

#### Changed Files
- `path/to/file`：改动说明。

#### Decisions
- 决策：为什么这样做。

#### Validation
- 运行命令：`{{COMMAND}}`
- 验证结果：{{PASS / FAIL / PARTIAL}}

#### Next
- 下一步 1。

#### Blockers
- 阻塞问题和需要用户确认的问题。
